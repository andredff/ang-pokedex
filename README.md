## Instalação
Para iniciar o **Frontend** do Angular utilize os comandos:
```bash
npm install
npm start
```
Assim que o processo terminar, automaticamente será aberta no seu navegador a página `localhost:4200`.  


# Testes
Para Iniciar os testes utilize os comandos:
```bash
ng teste
ng e2e
```
